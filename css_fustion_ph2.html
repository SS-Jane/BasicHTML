Responsive Wed design

การออกแบบเว็บที่ตอบสนองและแสดงผลได้ดีบนอุปกรณ์ต่างๆหรือขนาดหน้าจอที่หลากหลาย เช่น จอคคอมพิวเตอร์ หรือ สมาร์ทโฟน เป็นต้น

CSS tailwindcss

คำสั่งตรวจสอบขนาดหน้าจอ จะใช้ Media Query

Media Query คือ รูปแบบการเขียน style ให้แสดงผลตามขนาดหน้าจอที่แตกต่างกัน

Flexbox

คือเครืองมือใน CSS ที่ช่วยให้การจัดการ element ต่างๆ ในหน้าเว็บมีความง่ายและยืดหยุ่นมากยิ่งขึ้น

องค์ประกอบโดยใช้ Container และกล่อง 3 กล่อง

การจัดเรียงในแบบอื่นๆ

การกำหนด Flexbox

กำหนดทิศทางด้วย Flex-direction

การกำหนดขนาดด้วย Flex-wrap คือการห่อหุ่ม element

Flex Item

การกำหนด Propertise ให้กับ Item ใน Flexbox

Flex-basic กำหนดค่าความยืดหยุ่นเริ่มต้น

Flex-shrink หดเป็นกี่เท่าถ้าเทียบกับ item อื่น

Flex-grow ขยายเป็นกี่เท่าถ้าเทียบกับ Item อื่น

Flex Justify

การจัดวางตำแหน่งของ Item เทียบกับแกนหลัก ของ Container

Flex Alignment (Item)

การจัดวาง Item เทียบกับแกนตรงข้าม

Grid layout

การจัด Layout แบบ 2 มิติ เช่น ตาราง แตกต่างจาก Flexbox ที่เป็นแบบ 1 มิติ เช่นเรียงจากบนลงล่าง ซ้ายไปขวา
